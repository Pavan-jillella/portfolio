# Index Funds vs. Stock Picking: A Developer's Framework

As developers, we're trained to optimize. But when it comes to investing, optimization often backfires.

## The Developer's Bias

We think in systems. We see patterns. We believe we can build a better model. This makes us dangerous investors.

```typescript
// What developers think investing looks like
function pickStocks(market: Stock[]): Portfolio {
  const undervalued = market.filter(stock => stock.peRatio < 15);
  const growing = undervalued.filter(stock => stock.revenueGrowth > 0.2);
  return optimize(growing);
}

// What investing actually looks like
function investWisely(income: number): Portfolio {
  const monthlyInvestment = income * 0.25;
  return buyIndexFund("VTSAX", monthlyInvestment); // every month, forever
}
```

## The Data

Over any 30-year period since 1926, the S&P 500 has never lost money. Not once. The average annual return is roughly 10%.

Meanwhile, 90% of actively managed funds underperform their benchmark over 15+ years.

## Why Boring Wins

### Compound Interest

The math is simple but powerful:

- $500/month at 10% annual return
- After 30 years: ~$1,130,000
- Total invested: $180,000
- Returns: $950,000

The market does the heavy lifting. You just need to show up.

### Time in Market vs. Timing the Market

Missing the 10 best days in the market over 20 years cuts your returns by more than half. Nobody can consistently predict those days.

## My Approach

1. **Max out tax-advantaged accounts** — 401k, Roth IRA
2. **Total market index funds** — VTSAX or equivalent
3. **Automatic contributions** — set and forget
4. **Rebalance annually** — keep allocation on target
5. **Ignore the noise** — markets fluctuate, stay the course

## The Engineer's Edge

Where developers actually have an edge: building systems that automate good financial behavior. Automate savings. Automate investments. Remove emotion from the equation.

The best investment strategy is the one you'll stick with for decades. For most people, that's a boring index fund.
